## 📌기능 소개
회원가입/로그인  
- 회원가입시 관리자와 유저를 구분하여 하나의 디비에 저장합니다.
- 로그인시 관리자와 유저가 구별되어 페이지로 이동합니다.
  
마이페이지  
- 정보수정, 회원퇄퇴가 가능하며 회원탈퇴시 물리삭제로 구현하였습니다.

## 📌기술스택
스프링 프레임워크를 기본으로 MVC구조, 데이터베이스는 Oracle을 이용했습니다.
