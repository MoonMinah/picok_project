# picok_project
사진을 주제로 한 자유로운 커뮤니티 프로젝트입니다.

관리자 페이지에서 유저 페이지의 기능 수행을 관리할 수 있도록 유저페이지/관리자페이지 별도 분리했습니다.  
스프링 프레임워크를 기본으로 MVC구조, 데이터베이스는 Oracle을 이용했습니다.
